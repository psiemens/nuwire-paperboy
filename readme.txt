Allows posts to be automatically sent to the National University Wire.

== Description ==

This plugin provides an easy way for NUW members to submit content to the our online wire service. A valid NUW account and membership is required. 

== Installation ==

Intstallation Instructions

1. Upload `nuw-paperboy.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Enter your NUW account information on the 'NUW Settings' page.
